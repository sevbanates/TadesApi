# TadesApi
Web API for business portal

## Proje Yapısı

- **TadesApi.Portal**: Ana Web API projesi
- **TadesApi.BusinessService**: İş mantığı servisleri
- **TadesApi.Core**: Temel sınıflar ve yardımcı fonksiyonlar
- **TadesApi.Db**: Veritabanı entity'leri ve DbContext
- **TadesApi.Models**: View modelleri ve DTO'lar
